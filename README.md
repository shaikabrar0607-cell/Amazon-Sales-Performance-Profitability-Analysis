# Amazon Sales Performance & Profitability Analysis

## Project Overview

This project analyzes Amazon product sales data to identify revenue opportunities, customer engagement patterns, pricing effectiveness, and profitability drivers.

## Business Problem

Amazon manages thousands of products across multiple categories. Understanding which categories generate the highest engagement and revenue can help optimize pricing, promotions, and product strategies.

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

Amazon Product Sales Dataset from Kaggle

Dataset Size:
- 1,465 Products
- 16 Features

## Analysis Performed

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Revenue Analysis
- Discount Analysis
- Customer Rating Analysis
- Category Performance Analysis
- Product Portfolio Analysis

## Key Insights

- Electronics and Computer Accessories dominate product listings.
- Customer engagement strongly influences revenue potential.
- Heavy discounts do not always improve customer ratings.
- Revenue is concentrated within a few major categories.
- High-rated products present strong growth opportunities.

## Business Recommendations

- Optimize category-level discount strategies.
- Focus marketing efforts on high-performing products.
- Improve visibility of high-rated products with lower engagement.
- Reduce unnecessary discount leakage.
- Prioritize inventory planning for top-performing categories.
